## Installation

Refer to [Astro - Getting Started](https://docs.astro.build/en/getting-started/) page

```bash
$ npm create astro@latest
# $ pnpm create astro@latest
# $ yarn create astro

$ npm install gyg-wc
# $ pnpm add gyg-wc
# $ yarn add gyg-wc
```
